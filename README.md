# 🌿 Explainable Open-Set Plant Disease Diagnostic System

An Explainable AI-based agricultural decision support system for plant disease diagnosis using Vision Transformers and Open-Set Recognition.

## Features

- Known Disease Detection
- Unknown Plant Anomaly Detection
- Explainable AI (Attention Rollout)
- Disease Severity Estimation
- Recommendation Engine
- PDF Diagnostic Report
